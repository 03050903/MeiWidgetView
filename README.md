# MeiWidgetView
一款汇总了郭霖，鸿洋，以及自己平时收集的自定义控件集合库

## 引入

### Step 1. Add the JitPack repository to your build file

root gradle

````
    allprojects {
        repositories {
            ...
            maven { url "https://jitpack.io" }
        }
    }

````

### Step 2. Add the dependency

app gradle 

````
	dependencies {
	        compile 'com.github.HpWens:MeiBaseModule:1.0.5'
	}
````
